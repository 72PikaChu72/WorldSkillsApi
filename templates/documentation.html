API Документация

Общее

Это API позволяет делиться файлами между пользователями и управлять доступами к ним

Все запросы кроме /registration и /authorization требуют передачи токена в заголовке Authorization в формате

Bearer your_token

Регистрация пользователя

POST /registration

Зарегестрировать нового пользователя и получить его токен

Request Body:

{
    "email": "newuser@example.com",
    "password": "yourpassword",
    "first_name": "First",
    "last_name": "Last"
}

Ответ:

{
    "success": true,
    "message": "Success",
    "token": "your_token"
}

Авторизация

POST /authorization

Авторизовать пользователя и получить токен

Request Body:

{
    "email": "user@example.com",
    "password": "yourpassword"
}

Ответ:

{
    "success": true,
    "message": "Success",
    "token": "your_token"
}

Управление файлами

GET /files/diskpage

Получить список файлов пользователя.

Ответ:

[
    {
        "file_id": "123",
        "name": "file1.txt",
        "url": "/files/123",
        "accesses": [
            {
                "fullname": "Owner Name",
                "email": "owner@example.com",
                "type": "author"
            },
            {
                "fullname": "Co-Author Name",
                "email": "coauthor@example.com",
                "type": "co-author"
            }
        ]
    },
    {
        "file_id": "124",
        "name": "file2.txt",
        "url": "/files/124",
        "accesses": []
    }
]

GET /files/sharedpage

Получить список файлов доступных пользователю.

Ответ:

[
    {
        "file_id": "125",
        "name": "shared_file.txt",
        "url": "/files/125"
    }
]

POST /files

Upload a new file.

Request: Загруженный файл в multipart/form-data.

Ответ:

{
    "success": true,
    "message": "Success",
    "name": "file_name.txt",
    "url": "/files/123",
    "file_id": 123
}

DELETE /files/{fileid}

Удалить файл по id.

Ответ:

{
    "success": true,
    "message": "Deleted"
}

Доступ к файлам

POST /files/{fileid}/accesses

Выдать доступ к файлу пользователю используя его email.

Request Body:

{
    "email": "coauthor@example.com"
}

Ответ:

[
    {
        "fullname": "Owner Name",
        "email": "owner@example.com",
        "type": "author"
    },
    {
        "fullname": "Co-Author Name",
        "email": "coauthor@example.com",
        "type": "co-author"
    }
]

DELETE /files/{fileid}/accesses

Забрать доступ к файлу пользователя используя его email.

Request Body:

{
    "email": "coauthor@example.com"
}

Ответ:

[
    {
        "fullname": "Owner Name",
        "email": "owner@example.com",
        "type": "author"
    }
]
